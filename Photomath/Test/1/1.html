Ordered pair: Pair of elements enclosed in circular bracket is called as Ordered pair.

For example, (a,b) or (x,y).

Where as first element is a and second element is b.

Cartesian Product: If \textcolor{primary}{A} and \textcolor{secondary}{B} are two non-empty set their product is denoted by \textcolor{primary}{A} \times \textcolor{secondary}{B} and defined as:

﻿

\textcolor{primary}{A}\times \textcolor{secondary}{B} = \{ (x,y) | x\in \textcolor{primary}{A} \land y\in \textcolor{secondary}{B}\}

Relation: A subset of cartesian product \textcolor{primary}{A}\times \textcolor{secondary}{B} is called binary relation.

It is denoted by \textcolor{tertiary}{R}.

\textcolor{tertiary}{R} = \{ (x,y) | x\in \textcolor{primary}{A} \land y\in \textcolor{secondary}{B}\}

The given relation and sets \textcolor{tertiary}{R}, \textcolor{primary}{A}, \textcolor{secondary}{B} are given by:

\textcolor{tertiary}{R}=\{(0,7),(4,8),(7,0),(8,4)\}
\textcolor{primary}{A}=\{0,7,4,8\}
\textcolor{secondary}{B}=\{7,8,0,4\}

(a) For given relation, the domain (all first element of order pairs forming a relation) is given by:

\{0,7,4,8\}

(b) For given relation, the range (all second element of order pairs forming a relation) is given by:

\{7,8,0,4\}

(c) Function: The binary relation is called as function if it is:

1. Subset of \textcolor{primary}{A}\times \textcolor{secondary}{B}

2. Domain is equal to set A.

3. First elements of order pairs is not repeating.

So Cartesian Product is given by:

\textcolor{primary}{A}\times \textcolor{secondary}{B} =\begin{gathered} \{(0,7),(0,8),(0,0),(0,4),(7,7),\\(7,8),(7,0),(7,4),(4,7),(4,8),\\(4,0),(4,4),(8,7),(8,8),(8,0),(8,4)\}\end{gathered}

So, given relation is a function beacuse:

1. \textcolor{tertiary}{R}\subset \textcolor{primary}{A}\times \textcolor{secondary}{B}

2. Domain of \textcolor{tertiary}{R}= \textcolor{primary}{A}\

3. First elements of order pairs is not repeating.

SOLUTION

(a) Domain=\{0,7,4,8\}

(b) Range=\{7,8,0,4\}

(c) Given relation is a function.
